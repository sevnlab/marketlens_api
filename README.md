## 📌 Project Overview
이 프로젝트는 **백엔드 서비스**를 구축하기 위한 실무형 Spring Boot 프로젝트입니다.  
안정적이고 확장 가능한 REST API 서버를 목표로 합니다.

---

## 🛠 Environment
- **Java:** 21
- **Spring Boot:** 3.5.6
- **Build Tool:** Gradle 8.10.2
- **Database:** Oracle 19c
- **Cache:** Redis 7.2.0
- **Message Broker:** Apache Kafka 3.x (spring-kafka, 버전은 Spring Boot 3.5.6 BOM 자동 관리)

---

## 🔑 Features
- REST API 설계 및 구현
- Redis 캐싱 적용
- Oracle 19c 연동
- 예외 처리 및 로깅
- Swagger 문서화
- Kafka 기반 대기열 시스템 ([상세 문서](docs/WAITING_QUEUE.md))
